16:9 반응형 비디오 구현하기

iframe을 감싸서 ratio 역할을 하는 div를 하나 만든다
padding요소의 기준 값은 width임 => padding-top을 56.25%로 주면 16:9 비율 구현.
height:0을 주게 되면 박스 모델 높이에 영향을 주는 속성이 padding-top 밖에 없게 된다
iframe은 width, height: 100% 속성을 주어서 div 비율에 맞춰 반응형이 되도록

.video-ratio {
position: relative;
padding-top: 56.25%;
height: 0;
} .video-ratio iframe {
position: absolute;
top: 0;
left: 0;
width: 100%;
height: 100%;
}
